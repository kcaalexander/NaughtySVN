What Is NaughtySVN?

NaughtySVN (NSVN) is a easy-to-use GUI based Subversion client which works as an extension to Nautilus File Manager (Graphical shell for the GNOME desktop environment). Inspired by TortoiseSVN, Its an effort primarily for non-technical people to use subversion. NSVN has a graphical interface which can be activated from Nautilus file browsing window. Its graphical windows are designed to be simple and easy to use, which will allow users to create and administer subversion repository, checkout and commit working copies, handling working copies, browsing repositories and working copies, handling remote repositories and working copies etc. The software will be released under an GNU GENERAL PUBLIC LICENSE (GPL).

NaughtySVN's Immediate Features

Immediate list features is to develop a reasonably good framework and to test capabilities of Nautilus before source code is made available to other developers. NaughtySVN 0.0.1 release is expected to contain all the below features.

  • Easy-to-use GUI interface.
  • Integrating with Nautilus.
  • Show and Hide context menu items.
  • Subversion repository creation.
  • Checkout Subversion repository to working copy.
  • Commit working copy to Subversion repository.
  • Information on repository and working copy as properties.
  • GConf to store NSVN configuration and user preference.
  • Visually mark file and folder icons with emblems, to show status of working copies and repositories.
