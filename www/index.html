What Is NaughtySVN?

NaughtySVN (NSVN) is an easy-to-use Subversion client and has a Graphical User Interface. NSVN works as an extension to the Nautilus File Manager (a graphical shell for the GNOME desktop environment). The inspiration of this project is TortoiseSVN. The aim of this project is to help users who are not technically oriented to use Subversion.

NSVN has a graphical interface which can be activated from the Nautilus file browsing window. The graphical interface in NSVN has been designed to be simple and user friendly. Users will be able to perform operations such as administer a Subversion repository, checkout and commit working copies, handle working copies and so on. NaughtySVN will be released under a Apache/BSD-style open source license.

NaughtySVN's Immediate Features

Immediate list features is to develop a reasonably good framework and to test capabilities of Nautilus before source code is made available to other developers. NaughtySVN 0.0.1 release is expected to contain all the below features.

  • Easy-to-use GUI interface.
  • Integrating with Nautilus. (completed)
  • Show and Hide context menu items. (completed)
  • Subversion repository creation. (completed)
  • Checkout Subversion repository to working copy. (completed)
  • Commit working copy to Subversion repository. (under-devlopment)
  • Information on repository and working copy as properties. (under-devlopment)
  • GConf to store NSVN configuration and user preference. (under-devlopment)
  • Visually mark file and folder icons with emblems, to show status of working copies and repositories. (under-devlopment)
